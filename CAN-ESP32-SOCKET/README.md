

# CAN - ESP32 - SOCKET

This project directory sets up an ESP32 to receive CAN messages over SPI from an MCP2515 controller/transceiver module, and then send that data to a server over a TCP socket.

## How to use

Install the ESP-IDF extension in VSCode (or just use ESP-IDF, up to you).
Clone this repo into your directory of choice.
```
git clone -b laithalkinani https://github.com/laithalkinani/CAN-Telemetry.git
```
